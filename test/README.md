## Installation

To test the project go to the `root` of `(cds-ts-dispatcher)` and run :

```bash
npm install
```

### Run unit/integration

Go to the `root` folder and run :

```bash
npm run test:unit
```

### Run e2e tests

Go to the `root` folder and run :

```bash
npm run test:all
```

Check if all tests are completed.
